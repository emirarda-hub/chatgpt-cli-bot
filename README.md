# ChatGPT CLI Bot

A simple command-line chatbot built with Python and the OpenAI API.  
This project allows you to interact with an AI assistant directly from your terminal.

## Features

- Simple CLI interface
- Uses the OpenAI API
- Customizable parameters (temperature, tokens, etc.)
- Lightweight and easy to run

## Requirements

- Python 3.8+
- OpenAI API key
- requests library
